# Final Submission Zip Checklist

The teacher's screenshot asks for a top-level directory with the group name. For a solo project, use a clear name such as:

```text
AirDesk_Caden_Calderon/
```

## Required Items

- [ ] Final report PDF.
- [ ] README with:
  - [ ] GitHub/repo link.
  - [ ] video links.
  - [ ] prototype run instructions.
  - [ ] notes that help grading.
- [ ] LaTeX source directory.
- [ ] `PDFs-LiteratureSurvey/` directory with literature PDFs.
- [ ] Source code directory, or a text file linking to GitHub if the source directory does not upload cleanly.
- [ ] Short project video, ideally 3-5 minutes.
- [ ] Presentation video, ideally around 12 minutes and below 30 minutes.
- [ ] Programming/code walkthrough video.

## Suggested Final Zip Shape

```text
AirDesk_Caden_Calderon/
  Final_Report.pdf
  README.md
  Latex_Source/
  PDFs-LiteratureSurvey/
  Source_Code_or_GitHub_Link.txt
  Videos/
```

## README Must Say

- What AirDesk is.
- What is implemented.
- How to run safe replay/demo commands.
- Which commands require webcam or Hyprland.
- Where the paper source is.
- Where the videos are.
- What pilot evidence was collected.
- Any known limitations.
