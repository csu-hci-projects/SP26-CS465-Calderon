# PDFs-LiteratureSurvey

This folder contains the paper PDFs used for the AirDesk related-work pass.
The collection focuses on open, arXiv, ACM/author-hosted, or otherwise
available paper PDFs.

The PDFs are kept as individual files so each bibliography entry can be matched
to its source. Website/documentation sources, such as Hyprland documentation,
are cited in the paper but intentionally excluded from this folder.
